# AI-Based Attendance Management System 1.0

Initial prototype of a face recognition attendance system using OpenCV and basic facial feature matching.

## Features

- Face detection using Haar cascades
- Basic face recognition with LBPH algorithm
- Attendance logging to CSV
- Simple web interface for enrollment
- Camera feed preview

## Tech Stack

Python, OpenCV, Flask, NumPy, CSV

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/AI-based-Attendance-Management-System-1.0.git
cd AI-based-Attendance-Management-System-1.0
pip install -r requirements.txt
```

### Usage

```bash
python app.py
```
